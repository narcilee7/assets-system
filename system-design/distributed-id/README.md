# Distributed ID Generator

## 目标

训练分布式环境下全局唯一 ID 的生成策略、趋势递增保障、时钟回拨处理和性能优化。

## 核心问题

- Snowflake 的 bit 位怎么分配？
- 时钟回拨如何检测和应对？
- 号段模式（Segment）vs Snowflake 怎么选？
- 趋势递增和严格递增的区别？
- 高并发下如何降低 ID 生成延迟？
- 多机房部署时 worker id 如何分配？

## 面试追问

- 如果机器时钟发生大幅回拨，Snowflake 还能用吗？
- 如果 ID 需要暴露给用户，如何防止被遍历？
- 如果业务量增长 100 倍，号段模式如何平滑扩容？
- 如何在没有 NTP 的环境下保证 ID 唯一性？
