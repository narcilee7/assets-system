## 系统设计补齐方向

- 秒传：基于文件 hash 查询已有资源。
- 断点续传：查询已上传分片。
- 并发控制：限制同时上传分片数。
- 失败重试：单分片重试和整体恢复。
- 幂等：分片编号、文件标识和合并请求防重复。
- 存储：本地、对象存储、临时分片清理。
- 观测：上传耗时、失败率、分片错误分布。
