# Frontend Frameworks

Framework 能力要求达到“能用、能解释、能优化、能取舍、能局部实现”的水平。

## 框架矩阵

| 框架 | 核心能力 | 架构师关注 |
| --- | --- | --- |
| React | Fiber、Hooks、Concurrent、Suspense、RSC | 调度、渲染、状态、服务端组件 |
| Vue | 响应式、compiler、scheduler、Composition API | 依赖追踪、模板编译、组件组织 |
| Angular | DI、RxJS、Change Detection、Router | 大型企业应用和约束 |
| Next.js | SSR、SSG、ISR、RSC、Route Handler | 全栈渲染和缓存 |
| Nuxt | SSR、Nitro、server route、hydration | Vue 全栈架构 |
| Svelte | compile-time reactivity | 编译时优化 |
| Mini Framework | reactive、vdom、scheduler、router | 框架原理训练 |
