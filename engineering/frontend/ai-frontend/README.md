# AI Frontend

AI Frontend 训练 AI 原生交互：流式输出、工具调用状态、可中断任务、可回放事件和人机协作。

## 核心模块

| 模块 | 关键点 |
| --- | --- |
| Streaming Renderer | delta、structured event、markdown、code block |
| Tool State UI | pending、running、success、error、retry |
| Human-in-the-loop | confirmation、edit args、approve / reject |
| Session Timeline | event replay、trace、debug |
| Cancellation | AbortController、server cancel、UI rollback |
| Multimodal | image、file、voice、preview |
| Generative UI | AI |
