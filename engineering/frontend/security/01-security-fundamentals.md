# 前端安全基础

## 1. OWASP Top 10（前端相关）

| 排名 | 威胁 | 前端影响 |
|------|------|----------|
| A01 | 失效的访问控制 | 隐藏路由绕过、前端权限校验不足 |
| A03 | 注入 | XSS、HTML 注入、DOM 注入 |
| A07 | 身份识别与认证失效 | JWT 泄露、Cookie 劫持、弱密码策略 |
| A08 | 软件和数据完整性故障 | 供应链攻击、恶意 npm 包 |
| A10 | 服务端请求伪造（SSRF） | 前端作为跳板、URL 参数滥用 |

## 2. 信任边界

```
┌─────────────────────────────────────────────────────────────┐
│                        攻击者控制区                          │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐                  │
│  │ 恶意请求  │  │ 钓鱼页面  │  │ XSS Payload│              │
│  └────┬─────┘  └────┬─────┘  └────┬─────┘                  │
└───────┼─────────────┼─────────────┼─────────────────────────┘
        │             │             │
        ▼             ▼             ▼
┌─────────────────────────────────────────────────────────────┐
│                        浏览器沙箱                            │
│  ┌──────────────────────────────────────────────────────┐  │
│  │                   你的网站                            │  │
│  │  ┌──────────┐  ┌──────────┐  ┌──────────┐           │  │
│  │  │ DOM / JS  │  │ localStorage│ │ Cookie   │          │  │
│  │  └──────────┘  └──────────┘  └──────────┘           │  │
│  └──────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│                        服务端                               │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐                  │
│  │ API 接口  │  │ 数据库    │  │ 文件系统  │                  │
│  └──────────┘  └──────────┘  └──────────┘                  │
└─────────────────────────────────────────────────────────────┘
```

**核心原则**：
- 前端代码完全暴露，不要在前端做唯一的安全校验
- 所有安全校验必须在服务端重复验证
- 前端安全是深度防御的第一层，不是唯一一层

## 3. 威胁建模：STRIDE

| 威胁类型 | 描述 | 前端场景 |
|----------|------|----------|
| **S**poofing | 伪装身份 | 伪造 JWT、Session 固定 |
| **T**ampering | 篡改数据 | 修改 localStorage、DOM 篡改 |
| **R**epudiation | 否认行为 | 缺少操作日志 |
| **I**nformation Disclosure | 信息泄露 | 源码 map 泄露、错误信息暴露 |
| **D**enial of Service | 拒绝服务 | 无限循环、资源耗尽 |
| **E**levation of Privilege | 权限提升 | 前端权限绕过、越权访问 |

## 4. 攻击面分析

```
前端攻击面：
├── URL 参数（?search=<script>）
├── 表单输入
├── 文件上传（文件名、内容、元数据）
├── Cookie / localStorage / sessionStorage
├── WebSocket 消息
├── PostMessage（跨窗口通信）
├── 第三方脚本（广告、统计、CDN）
├── 浏览器扩展
├── Deep Link / URL Scheme
└── Service Worker
```

## 5. 安全头部速查

```
Content-Security-Policy: default-src 'self'

Referrer-Policy: strict-origin-when-cross-origin
Permissions-Policy: geolocation=(), microphone=(), camera=()
Strict-Transport-Security: max-age=31536000; includeSubDomains
```
