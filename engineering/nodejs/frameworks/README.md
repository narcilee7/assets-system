# Node.js Frameworks

## 框架矩阵

| 框架 | 适合场景 | 关注点 |
| --- | --- | --- |
| Express | 简单 API、生态成熟 | 约束弱、需要自建规范 |
| Koa | 轻量中间件、洋葱模型 | 生态和工程约束 |
| NestJS | 企业级服务、模块化、DI | 学习成本、装饰器 |
| Fastify | 高性能 API、schema-first | 插件模型、序列化 |
| Hono | Edge / lightweight | runtime 差异 |
| Next API / Route Handler | 全栈应用、BFF | cache、runtime、边界 |
