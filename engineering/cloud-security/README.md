# Cloud Security

云安全工程化训练 —— 达到"能设计云原生安全架构、能实施零信任网络、能保障多云环境数据安全"的水平。

## 训练哲学

1. **云的安全是共享责任**：云厂商负责基础设施安全，客户负责云上资源的安全配置。
2. **默认不安全**：云资源的默认配置往往过于宽松，需要显式加固。
3. **身份是新的边界**：VPC 边界已被打破，身份和权限成为核心安全控制点。
4. **一切皆代码**：基础设施即代码（IaC）让安全策略可版本化、可审计、可自动化。
