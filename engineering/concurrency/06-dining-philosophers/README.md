# 06 Dining Philosophers（哲学家就餐问题）

## 问题描述

5 位哲学家围坐在圆桌旁，每两位之间有一根筷子。哲学家的生活只有两个状态：
- **思考**：不需要筷子。
- **就餐**：必须同时拿到左右两根筷子。

要求实现一个调度策略，使得哲学家可以持续思考/就餐，且**不发生死锁**。

## 核心概念

- **死锁（Deadlock）**：每位哲学家都拿起左边的筷子，同时等待右边的筷子，永远等待。
- **资源排序**：给筷子编号，哲学家总是先拿编号小的筷子，再拿编号大的筷子，破坏循环等待条件。
- **限制同时就餐人数**：最多允许 4 位哲学家同时尝试就餐（信号量）。
- **非对称策略**：偶数编号哲学家先左后右，奇数编号哲学家先右后左。

## 约束

- 必须实现至少一种死锁避免策略。
- 不得让任何哲学家永远饿肚子（理想情况下）。

## 手写提示

1. 如果每位哲学家同时拿起左筷子，会发生什么？
2. 破坏死锁四条件中的哪一个最容易？
3. 如何验证没有死锁？（设置超时，如果所有哲学家在一段时间内都成功就餐过，则通过）

## 验证方式

```bash
make run
```

验证逻辑：每位哲学家必须成功就餐至少一定次数，且在有限时间内完成（无死锁）。
