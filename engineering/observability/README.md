# Observability

可观测性层训练 log、metric、trace 和调试能力。

## 核心构件

| 构件 | 状态 | 关键点 |
| --- | --- | --- |
| structured logger | todo | request_id、span_id、字段化 |
| metrics counter / histogram | todo | 延迟、错误率、吞吐 |
| trace context | todo | 跨函数 / 跨服务传播 |
| debug dump | todo | 状态快照 |
| health check | todo | liveness、readiness |
