# Backend Case Studies

案例用于把多个能力域串起来。每个案例都应该能从业务、接口、数据、一致性、可靠性和观测讲完整。

## 推荐案例

| 案例 | 关联能力 |
| --- | --- |
| File Upload Backend | API、幂等、分片状态、对象存储、断点续传 |
| Order Payment | 领域建模、事务、Saga、幂等、对账 |
| Task Queue | 消息语义、重试、DLQ、背压、观测 |
| Agent Tool Runtime | Tool、权限、幂等、Streaming、Eval |
| RAG Knowledge Service | indexing、ACL、retrieval、citation、eval |
