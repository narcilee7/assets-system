# Backend Security

后端安全训练认证、授权、租户隔离、审计和数据保护。

## 核心主题

| 主题 | 关键问题 |
| --- | --- |
| Authentication | 用户是谁 |
| Authorization | 用户能做什么 |
| Multi-tenancy | 数据和权限隔离 |
| Audit | 谁在什么时候做了什么 |
| Secret Management | 密钥存储、轮换、最小暴露 |
| Data Protection | PII、脱敏、加密、保留策略 |
