# Transformers

Transformer 是现代 LLM、VLM 和多模态模型的核心结构。

## 必会主题

| 主题 | 关键点 |
| --- | --- |
| Token Embedding | 离散 token 到向量 |
| Positional Encoding | absolute、relative、RoPE、ALiBi |
| Attention | Q/K/V、scaled dot-product、mask |
| Multi-head Attention | 多子空间表达 |
| FFN / MLP | 非线性变换和容量 |
| LayerNorm / RMSNorm | 训练稳定性 |
| Residual | 梯度和信息流 |
| Encoder / Decoder | BERT-like、GPT-like、Seq2Seq |
