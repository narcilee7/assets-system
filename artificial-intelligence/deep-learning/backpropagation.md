# 反向传播 (Backpropagation)

反向传播是深度学习的核心算法。它用链式法则高效计算损失函数对所有参数的梯度。

## 前向传播回顾

```
x → [Linear] → z = Wx + b → [ReLU] → a = max(0,z) → [Linear] → ŷ → Loss
```

## 链式法则

```
∂L/∂W₁ = ∂L/∂ŷ · ∂ŷ/∂a · ∂a/∂z · ∂z/∂W₁
```

从输出层向输入层逐层传播梯度。

## 具体推导示例

### 全连接层

```
z = Wx + b
∂L/∂W = ∂L/∂z · x^T
cache: 保存 x，用于计算梯度
```

### ReLU

```
a = max(0, z)
∂L/∂z = ∂L/∂a ⊙ (z > 0)    # 梯度在 z<0 处为 0
```

### Sigmoid

```
σ(z) = 1 / (1 + exp(-z))
∂σ/∂z = σ(z) · (1 - σ(z))
```

**梯度消失问题**：当 σ(z) 接近 0 或 1 时，梯度 ≈ 0，深层网络难以训练。

## 计算图与自动微分

```
现代框架（PyTorch、JAX）用计算图自动求导：

1. 前向：构建计算图，记录操作
2. 反向：从输出节点反向遍历图，应用链式法则
3. 累积：将多条路径的梯度相加
```

### PyTorch 示例

```python
x = torch.tensor([2.0], requires_grad=True)
y = x ** 2 + 3 * x
y.backward()
print(x.grad)  # dy/dx = 2x + 3 = 7
```

## 梯度检查 (Gradient Checking)

```
数值梯度 ≈ [L(θ+ε) - L(θ-ε)] / (2ε)

检查：||数值梯度 - 反向传播梯度|| / (||数值梯度|| + ||反向传播梯度||) < 1e-7
```

用于调试自定义层是否正确实现了 backward。

## 常见梯度问题

| 问题 | 现象 | 原因 | 解决 |
|------|------|------|------|
| 梯度消失 | 深层参数几乎不更新 | sigmoid/tanh 饱和 | ReLU、残差连接、更好的初始化 |
| 梯度爆炸 | loss → NaN | 梯度连乘累积 | 梯度裁剪、更小的学习率 |
| 死亡 ReLU | 部分神经元永久输出 0 | 负半区梯度为 0 | Leaky ReLU、PReLU |

## 常见误区

| 误区 | 正解 |
|------|------|
| 反向传播就是梯度下降 | ❌ 反向传播计算梯度，梯度下降用梯度更新参数 |
| 需要手动推导每层梯度 | ❌ 框架自动处理，但理解原理对调试至关重要 |
| 梯度消失只在深层网络出现 | ⚠️ 浅层也可能，取决于激活函数和初始化 |
