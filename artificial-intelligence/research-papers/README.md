# Research Papers

论文主线用于建立模型研究的阅读路径。

## 精读模板

```text
问题是什么？
已有方法缺什么？
核心方法是什么？
训练 / 数据 / 模型结构有什么变化？
实验如何证明有效？
局限是什么？
工程上如何迁移？
和其他论文的关系是什么？
```
