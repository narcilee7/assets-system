# 优化核心概念

优化是 AI 的训练引擎。从梯度下降到 AdamW，理解优化器的行为对训练稳定、收敛速度和最终性能至关重要。

## 梯度下降家族

### 批量梯度下降 (BGD)

```
θ = θ - η · ∇_θ L(θ; D_all)
```

- 稳定但慢，每次用全部数据

### 随机梯度下降 (SGD)

```
θ = θ - η · ∇_θ L(θ; x_i)
```

- 噪声大但逃离局部最优能力强，泛化通常更好

### 小批量梯度下降 (Mini-batch)

```
θ = θ - η · ∇_θ L(θ; B)    (B: batch size)
```

- 平衡了计算效率和更新稳定性
- **现代深度学习的默认选择**

## 动量方法

### Momentum

```
v_t = β · v_{t-1} + ∇_θ L(θ_t)
θ_{t+1} = θ_t - η · v_t
```

- 积累历史梯度方向，加速收敛，抑制震荡
- β 通常取 0.9

### Nesterov 加速梯度 (NAG)

```
v_t = β · v_{t-1} + ∇_θ L(θ_t - η · β · v_{t-1})
```

-  lookahead 版本，先按动量方向走一步再算梯度

## 自适应学习率

### AdaGrad

```
G_t = G_{t-1} + g_t²
θ_{t+1} = θ_t - η / √(G_t + ε) · g_t
```

- 对稀疏特征给大学习率
- 问题：学习率单调递减，可能过早停止

### RMSprop

```
G_t = β · G_{t-1} + (1-β) · g_t²
```

- 用指数移动平均替代累积，解决 AdaGrad 学习率消失问题

### Adam

```
m_t = β₁ · m_{t-1} + (1-β₁) · g_t        # 一阶矩（动量）
v_t = β₂ · v_{t-1} + (1-β₂) · g_t²       # 二阶矩（自适应）
m̂_t = m_t / (1-β₁^t)                      # 偏差修正
v̂_t = v_t / (1-β₂^t)
θ_{t+1} = θ_t - η · m̂_t / (√v̂_t + ε)
```

- 结合动量和自适应学习率
- 默认参数：β₁=0.9, β₂=0.999, ε=1e-8

### AdamW

```
θ_{t+1} = θ_t - η · (m̂_t / (√v̂_t + ε) + λ · θ_t)
```

- **权重衰减与 L2 正则解耦**
- Adam 的 L2 正则效果差，因为自适应学习率会抵消正则
- AdamW 直接对参数做衰减，效果更好
- **大模型训练的默认优化器**

## 学习率调度

| 策略 | 公式 | 使用场景 |
|------|------|---------|
| Step Decay | 每 N 轮乘一个因子 | 传统 CV |
| Cosine Annealing | `η_t = η_min + 0.5(η_max-η_min)(1+cos(πT_cur/T_max))` | 大模型预训练 |
| Warmup | 线性增加到目标 lr，再衰减 | Transformer 必备，防止早期梯度爆炸 |
| OneCycle | 先升后降 | 快速收敛 |

**大模型标配**：Warmup + Cosine Decay

## 二阶优化（简要）

| 方法 | 核心 | 限制 |
|------|------|------|
| Newton | 用 Hessian 逆矩阵 | O(n²) 参数，O(n³) 计算，不可行 |
| L-BFGS | 近似 Hessian | 适合小批量、全批量，不适合大模型 |

## 常见误区

| 误区 | 正解 |
|------|------|
| Adam 总是比 SGD 好 | ❌ SGD + Momentum 泛化通常更好，Adam 收敛快 |
| 学习率越小越好 | ❌ 太小收敛慢，太大震荡或不收敛 |
| 所有层用相同学习率 | ❌ Embedding 层通常需要更大 lr，预训练 vs 微调不同 |
| AdamW 就是 Adam + weight_decay | ⚠️ 实现细节不同，必须用 AdamW 类而非 Adam(l2=...) |
