# 概率与统计核心概念

AI 的本质是处理不确定性。概率论为模型提供不确定性建模的框架，统计学提供从数据中学习的方法。

## 核心分布

| 分布 | 概率密度/质量函数 | AI 应用 |
|------|------------------|---------|
| 正态分布 (Gaussian) | `p(x) = (1/√(2πσ²)) exp(-(x-μ)²/(2σ²))` | 噪声建模、权重初始化、VAE 隐空间 |
| 伯努利分布 | `p(x) = p^x (1-p)^(1-x)` | 二分类、Dropout |
| 分类分布 (Categorical) | `p(x) = ∏ π_i^{[x=i]}` | 多分类输出、token 采样 |
| 多项分布 | `p(x) = n!/(∏x_i!) ∏ π_i^{x_i}` | 词袋模型、主题模型 |

## 贝叶斯定理

```
P(θ|D) = P(D|θ) × P(θ) / P(D)
        后验      似然    先验     证据
```

**AI 视角**：
- **最大似然估计 (MLE)**：只优化 `P(D|θ)`，找到最可能产生数据的参数
- **最大后验估计 (MAP)**：优化 `P(D|θ) × P(θ)`，加入先验正则
- **贝叶斯神经网络**：维护参数的后验分布，而非点估计

## 期望、方差与协方差

| 概念 | 公式 | 含义 |
|------|------|------|
| 期望 E[X] | `∫ x p(x) dx` | 平均值 |
| 方差 Var(X) | `E[(X-μ)²]` | 离散程度 |
| 协方差 Cov(X,Y) | `E[(X-μ_x)(Y-μ_y)]` | 线性相关性 |

**BatchNorm 中的统计**：计算 batch 内特征的均值和方差，做标准化：
```
x̂ = (x - μ_B) / √(σ_B² + ε)
```

## 采样方法

| 方法 | 原理 | 应用 |
|------|------|------|
| Monte Carlo | 大数定律，用样本均值近似期望 | 强化学习价值估计、积分近似 |
| 重要性采样 | 从易采样分布估计难采样分布 | 策略梯度、离策略学习 |
| 拒绝采样 | 用包络函数过滤样本 | 简单分布变换 |
| Gibbs 采样 | 逐维度条件采样 | 图模型推断 |

## 假设检验与置信度

| 概念 | 说明 |
|------|------|
| p-value | 在原假设下观察到当前数据的概率 |
| 置信区间 | 参数估计的不确定性范围 |
| 统计功效 | 正确拒绝错误假设的概率 |

**AI 中的统计**：A/B 测试模型效果、评估指标差异是否显著。

## 常见误区

| 误区 | 正解 |
|------|------|
| 概率大 = 因果 | ❌ 相关 ≠ 因果 |
| 大数定律说样本均值 = 期望 | ⚠️ 是收敛，不是相等，且要求 i.i.d. |
| 正态分布适合所有数据 | ❌ 很多场景用对数正态、指数分布更合适 |
| p < 0.05 就是有效果 | ❌ 只是拒绝原假设，不代表实际意义大 |
