# 线性代数核心概念

线性代数是 AI 的数学语言。神经网络的所有操作——前向传播、反向传播、注意力计算——本质上都是矩阵运算。

## 核心对象

| 对象 | 定义 | AI 中的角色 |
|------|------|------------|
| 向量 (Vector) | 一维数组，表示方向和大小 | 词嵌入、特征向量、梯度 |
| 矩阵 (Matrix) | 二维数组 | 权重矩阵、注意力分数、协方差 |
| 张量 (Tensor) | 多维数组 | 批次数据、模型参数、激活值 |

## 关键运算

### 矩阵乘法

```
C = A × B    (A: m×n, B: n×p)  →  C: m×p
```

在 AI 中无处不在：
- 全连接层：`output = input @ W + b`
- 注意力：`Attention = softmax(Q @ K^T / √d) @ V`

### 转置 (Transpose)

`A^T`：行列互换。用于计算内积、协方差、注意力中的 K^T。

### 逆矩阵 (Inverse)

`A⁻¹`：满足 `A × A⁻¹ = I`。用于求解线性方程组、白化变换。

**注意**：大多数深度学习矩阵不可逆（非方阵或奇异）。伪逆 `A⁺` 更常用。

## 特征分解与 SVD

### 特征值分解 (EVD)

```
A = V Λ V⁻¹
```

- Λ：特征值对角矩阵，表示各方向上的缩放因子
- V：特征向量矩阵，表示主方向

### 奇异值分解 (SVD)

```
A = U Σ V^T
```

| 组件 | 含义 |
|------|------|
| U | 左奇异向量（行空间） |
| Σ | 奇异值（重要性排序） |
| V^T | 右奇异向量（列空间） |

**AI 应用**：
- PCA 降维：取前 k 个奇异值
- 低秩近似：压缩模型权重
- 推荐系统：矩阵分解

## 内积与外积

| 运算 | 输入 | 输出 | 含义 |
|------|------|------|------|
| 内积 (dot) | 两个向量 | 标量 | 相似度：`a·b = |a||b|cosθ` |
| 外积 (outer) | 两个向量 | 矩阵 | 生成秩-1 矩阵 |

**关键洞察**：注意力机制的核心就是内积——用内积衡量 query 和 key 的相似度。

## 常见误区

| 误区 | 正解 |
|------|------|
| 矩阵乘法满足交换律 | ❌ 不满足。`AB ≠ BA`，顺序影响结果 |
| 所有矩阵都有逆 | ❌ 只有满秩方阵可逆 |
| 张量就是高维矩阵 | ⚠️ 是，但要注意张量积和矩阵乘的区别 |
| SVD 只能用于方阵 | ❌ SVD 对任意矩阵都适用 |
