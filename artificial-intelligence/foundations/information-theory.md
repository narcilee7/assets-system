# 信息论核心概念

信息论为 AI 提供了量化不确定性和信息量的工具。从交叉熵损失到 KL 散度，从 VAE 到扩散模型，信息论是 AI 的理论基石之一。

## 信息量与熵

### 自信息 (Self-information)

```
I(x) = -log p(x)
```

- 事件越不可能，信息量越大
- 底为 2：单位 bit；底为 e：单位 nat

### 熵 (Entropy)

```
H(X) = -Σ p(x) log p(x)
```

- 随机变量的平均不确定性
- 均匀分布熵最大；确定性分布熵为 0

**AI 示例**：语言模型的 vocab 分布越"平坦"（各 token 概率接近），熵越高，模型越"不确定"。

## 交叉熵 (Cross-Entropy)

```
H(p, q) = -Σ p(x) log q(x)
```

- p：真实分布（one-hot 标签）
- q：模型预测分布

**为什么分类用交叉熵损失？**

1. 最大似然的等价形式：最小化交叉熵 = 最大化似然
2. 对错误预测惩罚大：当 q 接近 0 时，-log(q) → ∞
3. 梯度友好：相比 MSE，交叉熵在分类问题中梯度更大、收敛更快

## KL 散度 (Kullback-Leibler Divergence)

```
D_KL(p || q) = Σ p(x) log(p(x)/q(x))
             = H(p, q) - H(p)
```

- 衡量两个分布的差异
- **非对称**：`D_KL(p||q) ≠ D_KL(q||p)`
- 始终 ≥ 0，当且仅当 p=q 时为 0

**AI 中的应用**：

| 场景 | 用法 |
|------|------|
| VAE | `L = E[log p(x|z)] - D_KL(q(z|x) || p(z))` |
| 知识蒸馏 | 学生拟合教师的 softmax 分布 |
| RLHF | PPO 中的 KL penalty，防止策略偏离太远 |
| 变分推断 | 用易采样分布近似后验 |

## 互信息 (Mutual Information)

```
I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)
```

- 衡量两个变量的共享信息量
- 用于特征选择、表示学习、对比学习

## 困惑度 (Perplexity)

```
PPL = exp(H(p, q)) = exp(-1/N Σ log q(x_i))
```

- 语言模型的标准评估指标
- 几何平均的逆概率
- PPL = 100 等价于每次都从 100 个等概率选择中猜

## 常见误区

| 误区 | 正解 |
|------|------|
| 交叉熵和 KL 散度是一回事 | ❌ 差一个常数 H(p)，优化时等价但含义不同 |
| KL 散度是对称的 | ❌ 严格非对称，`D_KL(p\|\|q)` 要求 q 在 p 的支撑集上非零 |
| 熵越大越好 | ❌ 模型预测熵高 = 不确定，但标签熵高 = 任务难 |
| PPL 越低模型越好 | ⚠️ 通常是的，但过低的 PPL 可能过拟合 |
