# Alignment

Alignment 主线关注模型是否符合人类意图、是否安全、是否可靠。

## 核心目标

| 目标 | 含义 |
| --- | --- |
| Helpful | 真正解决用户问题 |
| Harmless | 不造成伤害 |
| Honest | 不知道时承认不确定 |
| Controllable | 可通过规则和上下文控制行为 |
| Auditable | 可解释、可追踪、可复盘 |

## 方法

- Preference data
- Constitutional AI
- Safety SFT
- RLHF / DPO
- Red teaming
- Refusal tuning
- Policy classifier
