# AI Safety

AI Safety 主线关注模型和应用在攻击、滥用、越权和不确定性下的表现。

## 主题

| 主题 | 关键点 |
| --- | --- |
| Jailbreak | 越狱、角色扮演、编码绕过 |
| Prompt Injection | RAG / tool 场景的上下文污染 |
| Data Exfiltration | secret、PII、tenant data |
| Tool Abuse | 高风险工具、越权调用 |
| Hallucination | 事实错误、引用错误 |
| Bias / Toxicity | 偏见、有害输出 |
| Policy | 分类器、拒答、人工确认 |
