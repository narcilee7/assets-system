# ML 评估指标

选择正确的评估指标是模型成功的关键。不同场景下，accuracy 可能完全误导。

## 分类指标

### 混淆矩阵

```
              预测正例    预测负例
实际正例      TP          FN
实际负例      FP          TN
```

### 基础指标

| 指标 | 公式 | 含义 | 适用场景 |
|------|------|------|---------|
| Accuracy | `(TP+TN)/(TP+TN+FP+FN)` | 整体正确率 | 类别平衡 |
| Precision | `TP/(TP+FP)` | 预测正例中实际正例的比例 | 误报代价高 |
| Recall | `TP/(TP+FN)` | 实际正例中被找出的比例 | 漏报代价高 |
| F1 | `2·P·R/(P+R)` | Precision 和 Recall 的调和平均 | 需平衡两者 |

### 多分类扩展

| 平均方式 | 计算方式 | 特点 |
|---------|---------|------|
| Macro | 每类 F1 取平均 | 平等对待每类，小类影响大 |
| Weighted | 按类别样本数加权 | 大类影响大 |
| Micro | 全局计算 TP/FP/FN | 等同于 accuracy（平衡数据） |

### ROC 与 AUC

- **ROC 曲线**：横轴 FPR (`FP/(FP+TN)`)，纵轴 TPR (`TP/(TP+FN)`)
- **AUC**：ROC 曲线下面积
  - AUC = 0.5：随机猜测
  - AUC = 1.0：完美分类
  - **含义**：随机选一个正例和负例，正例得分更高的概率

**何时不用 AUC？**
- 类别极度不平衡时，PR 曲线比 ROC 更有信息量
- 需要明确阈值时，看 Precision-Recall 曲线

## 回归指标

| 指标 | 公式 | 特点 |
|------|------|------|
| MSE | `1/N Σ(y-ŷ)²` | 对大误差惩罚重 |
| RMSE | `√MSE` | 与目标同量纲 |
| MAE | `1/N Σ|y-ŷ|` | 更鲁棒，对大误差不敏感 |
| MAPE | `1/N Σ|y-ŷ|/|y|` | 百分比，但 y=0 时不稳定 |
| R² | `1 - Σ(y-ŷ)²/Σ(y-ȳ)²` | 解释方差比例，1 为完美 |

## 排序指标

| 指标 | 定义 | 场景 |
|------|------|------|
| MAP | 平均 precision@k | 推荐、检索 |
| NDCG | 归一化折扣累积增益 | 考虑相关度等级的排序 |
| MRR | 第一个正确答案的倒数排名 | QA、搜索 |

## 校准 (Calibration)

```
模型预测概率 = 真实概率？
```

| 方法 | 原理 |
|------|------|
| 可靠性图 (Reliability Diagram) | 分桶看平均预测 vs 实际频率 |
| Platt Scaling | 用逻辑回归拟合温度参数 |
| Isotonic Regression | 单调非参数校准 |

**为什么重要？** 医学诊断、金融风控中，概率的绝对值和排序同样重要。

## 常见误区

| 误区 | 正解 |
|------|------|
| Accuracy 高 = 模型好 | ❌ 不平衡数据下 meaningless |
| F1 越高越好 | ⚠️ 需看 Precision 和 Recall 的具体值 |
| AUC 接近 1 就能上线 | ❌ 还需看低 FPR 区域的 TPR |
| 回归用 MSE 就够了 | ❌ 异常值多时 MAE 更稳健 |
