# 监督学习

监督学习是机器学习的核心范式：从带标签的数据中学习输入到输出的映射函数。

## 核心任务

| 任务 | 输出类型 | 损失函数 | 示例 |
|------|---------|---------|------|
| 回归 (Regression) | 连续值 | MSE、MAE、Huber | 房价预测、销量预测 |
| 二分类 (Binary) | 0/1 | BCE、Hinge | 垃圾邮件检测 |
| 多分类 (Multi-class) | 类别标签 | Cross-Entropy | 图像分类、情感分析 |
| 多标签 (Multi-label) | 标签集合 | Binary CE（每个标签独立） | 文章标签、疾病诊断 |

## 线性模型

### 线性回归

```
ŷ = w^T x + b
L = 1/N Σ (y_i - ŷ_i)²
```

- 解析解：`w = (X^T X)⁻¹ X^T y`
- 梯度下降：`w = w - η · X^T(Xw - y)`

### 逻辑回归

```
p(y=1|x) = σ(w^T x + b) = 1 / (1 + exp(-w^T x - b))
L = -Σ [y log(ŷ) + (1-y) log(1-ŷ)]
```

- 本质是线性分类器 + sigmoid
- 输出可解释为概率

### Softmax 回归

逻辑回归的多分类推广：

```
p(y=k|x) = exp(w_k^T x) / Σ_j exp(w_j^T x)
```

## 支持向量机 (SVM)

```
max margin:  1/2 ||w||² + C Σ ξ_i
subject to:  y_i(w^T x_i + b) ≥ 1 - ξ_i
```

| 类型 | 核函数 | 适用场景 |
|------|--------|---------|
| 线性 SVM | 无 | 高维稀疏数据（文本） |
| RBF SVM | `exp(-γ||x-x'||²)` | 中小规模，非线性边界 |
| 多项式 SVM | `(x^T x' + c)^d` | 图像等结构化数据 |

## 正则化

| 方法 | 公式 | 效果 |
|------|------|------|
| L1 (Lasso) | `λ Σ |w_i|` | 稀疏解，特征选择 |
| L2 (Ridge) | `λ Σ w_i²` | 权重衰减，防止过拟合 |
| Elastic Net | `λ₁|w| + λ₂w²` | 兼顾两者 |

**为什么 L1 产生稀疏性？**
- L1 的等高线是菱形，容易与损失函数的等高线在原点或坐标轴相交
- L2 的等高线是圆形，通常不会在坐标轴上

## 偏差-方差权衡

```
Expected Error = Bias² + Variance + Irreducible Noise
```

| 现象 | 原因 | 解决 |
|------|------|------|
| 欠拟合 (高偏差) | 模型太简单 | 增加特征、用更复杂模型 |
| 过拟合 (高方差) | 模型太复杂 | 正则化、更多数据、早停 |

## 常见误区

| 误区 | 正解 |
|------|------|
| 逻辑回归不能做非线性分类 | ❌ 通过特征工程（多项式、交叉特征）可以 |
| SVM 比神经网络好 | ❌ 大数据量下神经网络通常更强 |
| L1 总是比 L2 好 | ❌ 取决于是否需要特征选择 |
| 训练集准确率越高越好 | ❌ 可能是过拟合信号 |
