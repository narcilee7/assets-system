# Training Data

数据是模型能力的上限。训练数据主线关注数据来源、清洗、去重、质量、配比、合成和安全。

## 核心主题

| 主题 | 关键点 |
| --- | --- |
| Data Source | web、code、books、papers、QA、dialog |
| Cleaning | language filter、quality score、toxicity |
| Dedup | exact、near-dup、MinHash |
| Mixture | domain ratio、curriculum、upsampling |
| Synthetic Data | self-instruct、distillation、reasoning data |
| Contamination | benchmark leakage |
| Data Governance | license、PII、copyright |
