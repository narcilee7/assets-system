| 主题 | 关键点 |

| Objective | next token prediction、masked modeling |

| Stability | loss spike、nan、norm、initialization |
| Distributed Training | data parallel、tensor parallel、pipeline parallel、ZeRO |
| Checkpoint | save、resume、sharding |
