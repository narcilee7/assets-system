# Engineering Patterns

这一层训练 Java 工程模式的实现：创建型、结构型、行为型设计模式。

## 必会概念

- 单例模式：5 种写法，枚举单例最佳。
- 工厂模式：简单工厂、工厂方法、抽象工厂。
- 策略模式：封装算法族，让它们可以互相替换。
- 观察者模式：定义对象间的一对多依赖关系。
- 模板方法模式：定义算法骨架，子类实现步骤。
- 代理模式：静态代理、JDK 动态代理、CGLIB。
