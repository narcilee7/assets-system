# 数据结构与算法层

这一层不追求刷题数量，而是保留最有工程价值的一批结构和算法。

## 必会概念

- 哈希表适合快速查找和去重。
- 栈适合括号匹配、单调结构、DFS。
- 队列适合 BFS 和任务调度。
- 堆适合 Top K 和优先级队列。
- 图算法常用于依赖关系、任务编排和 pipeline。
