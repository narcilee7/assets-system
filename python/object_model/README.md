# 对象模型层

这一层训练 Python 的底层直觉：名字绑定、引用、可变对象、默认参数、类属性、对象拷贝和容器处理。

## 必会概念

- 变量不是盒子，是名字到对象的绑定。
- `is` 比较身份，`==` 比较值。
- 可变对象共享引用时，修改会被所有引用观察到。
- 默认参数只在函数定义时创建一次。
- 类属性先在类上查找，实例赋值后会形成实例属性。
- 浅拷贝只复制外层容器，深拷贝要处理嵌套对象、共享引用和循环引用。

## 易错点

`deep_copy` 不是简单递归。只递归会在循环引用上死循环，也会破坏原对象里的共享引用关系。核心动作是：创建副本后立刻放进 `memo`，再递归复制子对象。
