# Leadership

领导力不是职位，而是让正确的事情发生。

## 能力

| 能力 | 表现 |
| --- | --- |
| Ownership | 主动定义问题和结果 |
| Judgment | 做取舍，承担后果 |
| Influence | 让他人理解并愿意行动 |
| Accountability | 结果不好时能复盘和修正 |
| Talent Magnet | 吸引优秀的人一起做事 |
