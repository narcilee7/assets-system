# Management

管理能力用于通过团队拿结果。它不是控制人，而是建立目标、机制、反馈和成长系统。

## 管理主线

| 方向 | 关键问题 |
| --- | --- |
| Goal Setting | 目标是否清晰、可衡量、可拆解 |
| Hiring | 要什么人，如何判断 |
| Onboarding | 新人如何快速产生价值 |
| Feedback | 如何及时、具体、可行动 |
| Performance | 如何评估贡献和成长 |
| Delegation | 授权什么、检查什么 |
| Coaching | 如何让人变强 |
| Team Health | 信任、节奏、冲突、能量 |
