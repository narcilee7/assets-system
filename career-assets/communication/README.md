# Communication

沟通能力用于让复杂问题被理解、被支持、被推进。

## 场景

| 场景 | 目标 |
| --- | --- |
| Upward Communication | 让管理者理解风险、进展、选择 |
| Cross-team Alignment | 让多方对目标和边界达成一致 |
| Conflict | 处理分歧和优先级冲突 |
| Architecture Review | 清晰表达权衡和决策 |
| Incident Communication | 稳定、透明、事实优先 |
