# Interview System

面试系统用于把能力、经历和证据组织成稳定表达。

## 模块

| 模块 | 目标 |
| --- | --- |
| Resume | 证据密度 |
| Story Bank | STAR / CAR 故事 |
| Technical Depth | 深挖项目和架构 |
| Leadership | 影响力和团队协作 |
| System Design | 架构判断 |
| Negotiation | offer 取舍 |
