# Career Assets

职业资产主线用于把技术能力转化为长期职业复利。目标不是只成为更强的工程师，而是能在不同阶段切换角色：高级工程师、架构师、Tech Lead、Manager、创业者、技术合伙人。

## 职业资产模型

| 层级 | 能力 | 判断标准 |
| --- | --- | --- |
| L1 Execution | 稳定交付任务 | 能把事情做完、做对、可复盘 |
| L2 Ownership | 独立负责模块 / 项目 | 能定义问题、推进协作、承担结果 |
| L3 Technical Leadership | 技术方向和架构影响力 | 能带动多人做正确技术决策 |
| L4 People / Team Leadership | 团队目标、协作、反馈、成长 | 能通过团队拿结果 |
| L5 Business Leadership | 产品、商业、增长、客户、成本 | 能把技术连接到业务 |
| L6 Founder / Executive | 战略、融资、组织、市场、现金流 | 能构建公司或事业线 |

## 资产准入标准

职业资产必须能回答：

```text
我解决了什么问题？
我为什么是合适的人？
我如何影响他人或组织？
有什么可验证证据？
这个资产能否复用到面试、晋升、管理或创业？
```

## 推荐路线

```text
个人定位
-> 作品和证据链
-> 技术领导力
-> 团队管理
-> 产品和商业思维
-> 创业和资本
```
