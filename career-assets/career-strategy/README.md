# Career Strategy

职业战略用于回答：我是谁、我要去哪里、我选择什么战场、我积累什么不可替代资产。

## 核心问题

- 我的长期定位是 AI 全栈工程师、架构师、技术管理者、创业者，还是组合路径？
- 我的核心复利资产是什么？
- 哪些机会值得投入，哪些只是短期噪音？
- 1 年、3 年、5 年分别要达到什么状态？
