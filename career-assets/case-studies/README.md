# Career Case Studies

案例用于把职业选择、团队管理、技术领导和创业判断串起来。

## 推荐案例

| 案例 | 关注点 |
| --- | --- |
| From Engineer to Architect | 技术影响力和架构治理 |
| From IC to Manager | 角色切换、授权、反馈 |
| Technical Strategy Memo | 技术路线和业务价值 |
| Startup MVP | 用户、假设、验证 |
| Fundraising Story | 市场、团队、产品、数据 |
| Career Pivot | 机会成本和复利资产 |

## 模板

```text
背景
目标
约束
选择
行动
结果
复盘

```
