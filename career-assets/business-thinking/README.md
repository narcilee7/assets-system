# Business Thinking

商业思维用于判断技术是否能转化为价值。

## 核心主题

| 主题 | 问题 |
| --- | --- |
| Business Model | 谁付钱，为什么付钱 |
| Pricing | 如何定价，价值锚点是什么 |
| Unit Economics | 获客成本、毛利、留存 |
| Competition | 替代品和差异化 |
| Distribution | 如何触达客户 |
| ROI | 投入产出是否合理 |
