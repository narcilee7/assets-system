# Writing

写作是职业复利的放大器：它沉淀判断、复盘经验、建立影响力。

## 写作类型

| 类型 | 目标 |
| --- | --- |
| Technical Design | 推进方案和决策 |
| Postmortem | 复盘事实、根因和预防 |
| Learning Notes | 把学习转成可复用资产 |
| Public Article | 建立外部影响力 |
| Strategy Memo | 表达方向、取舍和路径 |
