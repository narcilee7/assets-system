# Networking

人脉不是社交数量，而是长期可信关系。

## 关系类型

| 类型 | 价值 |
| --- | --- |
| Peers | 信息、协作、机会 |
| Mentors | 反馈、判断、视野 |
| Sponsors | 推荐、背书、机会 |
| Customers | 真实需求和商业反馈 |
| Investors | 资本和市场视角 |
| Community | 影响力和分发 |
