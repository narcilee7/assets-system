# Startup

创业主线训练从 0 到 1：问题、用户、MVP、PMF、增长、团队、融资和现金流。

## 阶段

| 阶段 | 目标 |
| --- | --- |
| Problem Discovery | 找到真实痛点 |
| Customer Discovery | 验证用户和场景 |
| MVP | 最小可验证产品 |
| PMF | 留存、复购、口碑、付费 |
| Growth | 获客、转化、留存 |
| Organization | 合伙人、团队、文化 |
| Fundraising | 叙事、数据、投资人 |
