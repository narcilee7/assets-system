# Team Systems

团队系统关注机制，而不是靠个人英雄主义。

## 机制

| 机制 | 目标 |
| --- | --- |
| Planning | 目标、优先级、资源 |
| Execution Rhythm | 周会、站会、里程碑 |
| Review | 方案评审、代码评审、设计评审 |
| Quality | 测试、发布、回滚、事故复盘 |
| Knowledge | 文档、分享、决策记录 |
| Culture | 透明、反馈、责任、学习 |
