# Dynamic Programming

## 识别信号

- 求最值、计数、可行性。
- 选择会影响后续状态。
- 子问题重复出现。

## 四步

1. 定义状态。
2. 写出转移。
3. 确定初始化。
4. 确定遍历顺序。

## 常见类型

| 类型 | 例子 |
| --- | --- |
| 一维 DP | Climbing Stairs、Maximum Subarray |
| 背包 | Coin Change |
| 序列 DP | Longest Increasing Subsequence |
| 二维路径 DP | Unique Paths |
| 区间 DP | 戳气球、回文区间 |
