# Leetcode

## 链表

1. Leetcode 2. Add Two Numbers
难度：中等

描述：给定两个链表，表示两个非负整数。它们的每个节点包含一个数字，按逆序存储两个整数。求这两个数的和，返回一个新的链表。

2. Leetcode 21. Merge Two Sorted Lists
难度：简单

描述：将两个有序链表合并为一个新的有序链表。

3. Leetcode 141. Linked List Cycle
难度：简单

描述：给定一个链表，判断链表是否有环。如果链表有环，返回 true，否则返回 false。

4. Leetcode 160. Intersection of Two Linked Lists
难度：中等

描述：给定两个单链表，判断它们是否相交。若相交，返回相交的起始节点，否则返回 null。

5. Leetcode 19. Remove Nth Node From End of List
难度：中等

描述：给定一个链表，删除从末尾数起的第 n 个节点，返回修改后的链表。

6. Leetcode 2. Add Two Numbers
难度：中等

描述：给定两个链表，表示两个非负整数。它们的每个节点包含一个数字，按逆序存储两个整数。求这两个数的和，返回一个新的链表。

7. Leetcode 24. Swap Nodes in Pairs
难度：中等

描述：给定一个链表，交换相邻的两个节点。你必须在不修改节点值的情况下完成这项操作。

8. Leetcode 23. Merge k Sorted Lists
难度：困难

描述：给定 k 个有序链表，合并这些链表成为一个新的有序链表。

9. Leetcode 25. Reverse Nodes in k-Group
难度：困难

描述：给定一个链表，每 k 个节点反转一次。你需要在修改链表的结构时，保持节点之间的顺序。

10. Leetcode 206. Reverse Linked List
难度：简单

描述：反转一个单链表。
