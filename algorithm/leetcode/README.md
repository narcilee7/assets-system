| 分类 | 目录 | 对应模式 |
| --- | --- | --- |

| 链表 | `problems/chain/` | 快慢指针、递归、虚拟头节点 |
| 树 | `problems/tree/` | DFS、BFS、递归分治 |
| 图 | `problems/graph/` | DFS、BFS、克隆、连通性 |
| 动态规划 | `problems/dynamic_plan/` | 一维 DP、二维 DP、状态压缩 |
