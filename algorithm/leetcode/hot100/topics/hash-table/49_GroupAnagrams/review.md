# LeetCode 49 — Group Anagrams

## 模式
哈希 / 排序

## 识别信号
- 字母异位词分组
- 互为重排列
- 共享 key 的特性

## 边界
- 空字符串数组
- 全是空字符串
- 单一字符重复

## 实现要点
- 排序后作为 key（互为异位词排序后相同）
- 或计数后作为 key（26 个字母频次数组）
- Map 收集分组

## 复杂度
- 时间: O(n * k log k) k 为单词长度
- 空间: O(n*k)

## 追问
- 如果用计数作为 key？→ 数组转字符串
- Unicode 字符集？→ 使用 Map 计数
- 如果找字母异位词对？→ 双指针或哈希
