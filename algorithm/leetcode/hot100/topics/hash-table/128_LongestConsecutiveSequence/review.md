# LeetCode 128 — Longest Consecutive Sequence

## 模式
哈希 / 并查集

## 识别信号
- 最长连续序列（值连续递增）
- O(n) 时间要求
- 哈希表查找相邻元素

## 边界
- 空数组 → 0
- 有重复数字
- 单个元素

## 实现要点
- 放入哈希表
- 找连续序列起点（num-1 不存在）
- 从起点向后扩展计数
- O(n) 因为每个元素只访问一次

## 复杂度
- 时间: O(n)
- 空间: O(n)

## 追问
- 并查集怎么做？→ Union find 相邻元素
- 如果要求 O(1) 空间？→ 不可能，必须哈希
- 如果数组有序？→ 线性扫描 O(n)
