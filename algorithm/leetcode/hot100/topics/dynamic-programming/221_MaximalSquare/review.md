# LeetCode 221 — Maximal Square

## 模式
动态规划

## 识别信号
- 二维网格中找全是 1 的最大正方形
- 面积问题
- 状态依赖左上、左、上

## 边界
- 全 0 → 0
- 全 1 → min(m,n)^2
- 单行或单列

## 实现要点
- dp[i][j] = 以 (i,j) 为右下角的最大正方形边长
- dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1 if grid[i][j]=='1'
- 空间优化到 O(n)

## 复杂度
- 时间: O(m*n)
- 空间: O(n)

## 追问
- 最大矩形（1 面积）？→ 84 题单调栈
- 如果找最大长方形？→ 改 min 为其他逻辑
- 如果用前缀和？→ 二维前缀和 O(1) 查询每块
