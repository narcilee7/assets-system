# LeetCode 152 — Maximum Product Subarray

## 模式
动态规划（双状态）

## 识别信号
- 最大乘积子数组
- 乘法可能变负数
- 需要同时维护最大和最小

## 边界
- 全正数 → 全部乘积最大
- 全负数 → 最大（奇数个）或最小（偶数个）
- 含零

## 实现要点
- maxEnd[i] = max(maxEnd[i-1] * a[i], minEnd[i-1] * a[i], a[i])
- minEnd[i] = min(...)
- 全局 max 比较
- 零时重置

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 为什么同时要最小值？→ 负数乘负数变正
- 加法版本（最大和子数组）？→ Kadane，只维护一个
- 如果找最小乘积？→ 对称
