# LeetCode 198 — House Robber

## 模式
动态规划

## 识别信号
- 不能抢连续两家
- 选择最优
- 房屋排成一排

## 边界
- 空数组 → 0
- 一间房 → 金额
- 两间房 → max

## 实现要点
- dp[i] = max(dp[i-1], dp[i-2] + nums[i-1])
- 不抢第 i 间 或 抢（则不能抢 i-1）
- 空间优化到 O(1)

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 环形房屋？→ 分两种情况，第1间取或不取
- 二叉树版本？→ 不能抢父子节点
- 如果能抢 k 间？→ 状态扩展 dp[i][j]
