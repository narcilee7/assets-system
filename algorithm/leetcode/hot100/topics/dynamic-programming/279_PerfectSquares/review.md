# LeetCode 279 — Perfect Squares

## 模式
动态规划 / BFS / 数学

## 识别信号
- 最少完全平方数组成 n
- 完全背包
- 四平方和定理

## 边界
- n = 0 → 0
- n 是完全平方数 → 1
- n = 3 → 4（唯一不是 1/2/4 的情况）

## 实现要点
- DP: dp[i] = min(dp[i-j*j] + 1)
- BFS: 图论视角，每步减去平方数
- 数学: 四平方和定理（极少用到）

## 复杂度
- 时间: O(n * sqrt(n))
- 空间: O(n)

## 追问
- 四平方和定理？→ 1/2/3/4，四种答案
- 如果求所有表示？→ 回溯收集
- 如果限制最多 k 个平方数？→ DP 加一维
