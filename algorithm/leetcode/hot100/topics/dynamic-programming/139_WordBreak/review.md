# LeetCode 139 — Word Break

## 模式
动态规划

## 识别信号
- 字符串能否被字典中的单词分割
- 前缀子问题
- 完全背包

## 边界
- 空字符串 → true
- 单词比字符串长
- 无法分割

## 实现要点
- dp[i] = 字符串前 i 个能否被分割
- dp[i] = dp[j] && wordDict.contains(s[j:i]) for any j < i
- 从左到右填表

## 复杂度
- 时间: O(n * L) L 为字典中最长单词
- 空间: O(n)

## 追问
- 如何优化到 O(n)？→ 字典用 Set，遍历时检查
- 如果要求返回分割方式？→ 记录前驱
- 如果找最少单词数？→ DP 加计数
