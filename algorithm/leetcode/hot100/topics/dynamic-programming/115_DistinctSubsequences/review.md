# LeetCode 115 — Distinct Subsequences

## 模式
动态规划

## 识别信号
- s 中有多少种方式产生 t
- 子序列计数
- 重复字符处理

## 边界
- s 比 t 短
- s 和 t 相等
- s 或 t 为空

## 实现要点
- dp[i][j] = s 前 i 个产生 t 前 j 个的方式数
- s[i-1] == t[j-1]: dp[i][j] = dp[i-1][j-1] + dp[i-1][j]
- 否则: dp[i][j] = dp[i-1][j]
- dp[i-1][j] 表示不用 s[i-1] 匹配

## 复杂度
- 时间: O(mn)
- 空间: O(mn) 或 O(n) 优化

## 追问
- 为什么是加 dp[i-1][j]？→ s[i-1] 不参与匹配时，t[j-1] 仍需匹配
- 如果要输出具体方式？→ 回溯记录路径
- 如果 s 有重复字符？→ DP 本身已处理
