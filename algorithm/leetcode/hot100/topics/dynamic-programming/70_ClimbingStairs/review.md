# LeetCode 70 — Climbing Stairs

## 模式
动态规划（斐波那契变形）

## 识别信号
- 每步可选 1 或 2 → 只能从 i-1 或 i-2 到达 i
- 计数问题 → dp[i] = dp[i-1] + dp[i-2]
- 与斐波那契完全一致

## 边界
- n = 1 → 1
- n = 2 → 2
- n = 0（题目一般 n >= 1）

## 实现要点
- 滚动数组优化：只保留前两个状态
- 初始值 dp[1]=1, dp[2]=2

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 如果每次可以爬 1、2、3 步呢？→ dp[i] = dp[i-1] + dp[i-2] + dp[i-3]
- 如果要求输出具体路径呢？→ 回溯
- 矩阵快速幂优化到 O(log n)？
