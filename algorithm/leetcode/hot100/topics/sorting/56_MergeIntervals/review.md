# LeetCode 56 — Merge Intervals

## 模式
排序 / 合并区间

## 识别信号
- 区间重叠合并
- 多个区间交叉
- 按起点排序后线性扫描

## 边界
- 无重叠区间 → 直接返回
- 完全包含关系
- 首尾相接（如 [1,2] 和 [2,3]）

## 实现要点
- 按起点升序排序
- 当前区间与上一结果区间比较
- 更新 end 为 max(当前 end, 结果 end)

## 复杂度
- 时间: O(n log n)
- 空间: O(log n) 排序栈空间，O(1) 输出

## 追问
- 如果按终点排序？→ 也可以
- 如果求不重叠区间数？→ 类似贪心
- 如果有新的区间要插入？→ 先插入再合并
