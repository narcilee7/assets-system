# LeetCode 207 — Course Schedule

## 模式
拓扑排序 / DFS / BFS

## 识别信号
- 依赖关系有向图
- 判断是否存在环
- 课程先修关系

## 边界
- 无前置课程
- 直接环：1→2→1
- 间接环

## 实现要点
- BFS: 计算入度，删入度为 0 的节点
- DFS: 三色标记（白=未访问，灰=访问中，红=已完成）
- 有环时灰色节点会重复访问

## 复杂度
- 时间: O(V+E)
- 空间: O(V+E)

## 追问
- 如何返回修课顺序？→ 210 题
- 如果要判断是否能完成所有课程？→ BFS 后检查是否处理了所有节点
- 环检测的其他方法？→ 并查集（仅能检测特定环）
