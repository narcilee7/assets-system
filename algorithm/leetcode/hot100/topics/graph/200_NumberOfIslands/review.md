# LeetCode 200 — Number of Islands

## 模式
DFS / BFS / 并查集

## 识别信号
- 二维网格连通分量
- 4 方向遍历
- 计数连通区域

## 边界
- 空网格
- 全是 '1' 或全是 '0'
- 环形岛屿

## 实现要点
- 遍历每个格子，遇 '1' 则答案+1
- DFS/BFS 沉没连通的 '1'
- visited 或原地标记

## 复杂度
- 时间: O(m*n)
- 空间: O(m*n) 递归栈或队列

## 追问
- 并查集怎么做？→ 联合相邻的 '1'
- 如果求最大岛屿面积？→ 沉没时记录大小
- 如果是三维网格？→ 方向扩展到 6 个
