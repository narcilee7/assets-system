# LeetCode 301 — Remove Invalid Parentheses

## 模式
BFS / 回溯

## 识别信号
- 删除最少括号使字符串合法
- 求所有可能结果
- BFS 枚举删除位置

## 边界
- 原本合法
- 只有左括号或右括号
- 删除一个后合法

## 实现要点
- BFS: 每层删一个括号，检查是否合法，不合法继续下一层
- 回溯: 递归尝试删除每个括号，剪枝去重
- 用 set 去重

## 复杂度
- 时间: O(n * 2^n)
- 空间: O(n * 2^n)

## 追问
- 为什么 BFS 找到即最短？→ 按层删，第一层即最少删除
- 如何去重？→ set 或跳过连续相同括号
- 贪心不行？→ 局部最优不一定是全局最优
