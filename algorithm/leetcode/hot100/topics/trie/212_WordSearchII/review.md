# LeetCode 212 — Word Search II

## 模式
前缀树 / DFS / 回溯

## 识别信号
- 在二维网格中找多个单词
- 前缀树加速查找
- 剪枝回溯

## 边界
- 空网格或空单词列表
- 无匹配
- 单词重复

## 实现要点
- 建前缀树存储所有单词
- DFS 遍历网格，从前缀树分支查询
- 访问过的格子标记，防止重复
- 找到单词后从前缀树删除（剪枝）

## 复杂度
- 时间: O(m * n * 4^L) L 为单词最大长度
- 空间: O(total chars)

## 追问
- 为什么删除找到的单词？→ 避免重复和无效搜索
- 如果不用前缀树？→ 暴力查找每个单词 O(m*n*k)
- 如何优化？→ 剪枝和去重
