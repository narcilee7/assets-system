# LeetCode 472 — Concatenated Words

## 模式
前缀树 / DP

## 识别信号
- 哪些单词由其他单词拼接而成
- 多个短词拼成长词
- 类似单词拆分

## 边界
- 无可拼接单词
- 单词本身由自己拼接（排除）
- 嵌套拼接

## 实现要点
- 建前缀树存储所有单词
- DP: word 可拼接 = dp[j] && Trie 包含 word[j:]
- 排除自身：检查时不包含当前单词
- 先按长度排序（短词在前）

## 复杂度
- 时间: O(n * L^2) L 为单词长度
- 空间: O(total chars)

## 追问
- 为什么按长度排序？→ 短词必须先在字典中才能拼长词
- 与 139 Word Break 的区别？→ 字典是动态的，需先构建
- 如果要返回拼接方式？→ 记录 DP 路径
