# LeetCode 153 — Find Minimum in Rotated Sorted Array

## 模式
二分查找（变体）

## 识别信号
- 旋转有序数组找最小值
- 一半是有序的
- 最小值在旋转点

## 边界
- 未旋转（单侧有序）
- 旋转点在头或尾
- 单调

## 实现要点
- 二分 mid
- 如果 nums[mid] > nums[right]，最小值在右半
- 否则最小值在左半或 mid 本身

## 复杂度
- 时间: O(log n)
- 空间: O(1)

## 追问
- 如果有重复元素？→ 最坏 O(n)，退化为线性搜索
- 找最大值？→ 类似，在有序的一半找
- 如果找旋转点位置？→ 同算法
