# LeetCode 33 — Search in Rotated Sorted Array

## 模式
二分查找（变体）

## 识别信号
- 旋转有序数组查找
- 有一半是有序的
- 经典二分变体

## 边界
- 数组未旋转（单侧有序）
- target 在旋转点一侧
- 重复元素（最坏 O(n)）

## 实现要点
- 二分查找中判断哪半边有序
- 在有序半边中判断 target 范围
- 缩小搜索区间

## 复杂度
- 时间: O(log n)，最坏 O(n)
- 空间: O(1)

## 追问
- 如何判断重复元素导致最坏？→ 线性探测
- 如果找最小值？→ 153 题
- 如果返回下标不存在？→ -1
