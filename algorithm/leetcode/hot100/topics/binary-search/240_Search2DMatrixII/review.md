# LeetCode 240 — Search a 2D Matrix II

## 模式
二分 / 分治

## 识别信号
- 每行升序，每列升序的二维矩阵搜索
- 类似二分但二维
- 从角落开始

## 边界
- 空矩阵
- 单个元素
- 目标不在矩阵中

## 实现要点
- 从右上角或左下角开始
- 右上角：大于 target 往左，小于往下
- 类似二分排除行/列

## 复杂度
- 时间: O(m+n)
- 空间: O(1)

## 追问
- 二分怎么做？→ 对每行二分 O(m log n)
- 为什么从角落开始？→ 可以排除行列
- 如果矩阵不是每行首元素大于前一行尾元素？→ 240 可以，74 不行
