# LeetCode 1 — Two Sum

## 模式
哈希表（空间换时间）

## 识别信号
- 找两数之和为 target
- 数组无序，无法双指针排序后处理
- 需要 O(1) 查找 → Map

## 边界
- 无解时返回 []
- 同一个元素不能重复使用 → 先查后存
- 负数和零的情况

## 实现要点
- 一遍遍历：边存边查
- `map.has(diff)` 在 `map.set(nums[i], i)` 之前，防止重复使用同一元素

## 复杂度
- 时间: O(n)
- 空间: O(n)

## 追问
- 如果要求返回元素值而非下标呢？
- 如果数组有序怎么处理？（双指针）
- 如果要三数之和为 target 呢？（排序 + 双指针）
