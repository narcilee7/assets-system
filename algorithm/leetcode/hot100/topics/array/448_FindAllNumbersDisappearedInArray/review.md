# LeetCode 448 — Find All Numbers Disappeared in an Array

## 模式
原地哈希

## 识别信号
- 数组值在 [1,n]，有些缺失有些重复
- O(1) 空间要求
- 原地标记

## 边界
- 全缺失
- 全重复
- 完美情况（无缺失）

## 实现要点
- 遍历数组，将 nums[i] 对应的索引位置标记为负
- 再次遍历，未被标记的索引即缺失的数
- 注意：已负的不要再取绝对值

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 如果不能修改数组？→ 需要额外 O(n) 空间
- 如果数组值不在 [1,n]？→ 哈希表
- 如果要原地且不改变值？→ 交换法
