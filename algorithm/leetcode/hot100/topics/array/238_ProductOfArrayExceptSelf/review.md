# LeetCode 238 — Product of Array Except Self

## 模式
前缀和（变体）

## 识别信号
- 每个元素乘积除自身外
- 不能用除法
- 前缀积思想

## 边界
- 含零 → 零元素部分特殊处理
- 全零或多个零
- 负数

## 实现要点
- res[i] = prefix[i-1] * suffix[i+1]
- 两遍遍历构建前后缀
- O(1) 空间额外（输出数组外）

## 复杂度
- 时间: O(n)
- 空间: O(1) 不计输出数组

## 追问
- 如果有多个零？→ 所有位置除含多个零的位置外都是0
- 如果要求 O(1) 空间且不算输出数组？→ product of all 存起来，再算
- 除法限制的原因？→ 避免除零 + 精度问题
