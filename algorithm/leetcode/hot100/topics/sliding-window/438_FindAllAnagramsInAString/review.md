# LeetCode 438 — Find All Anagrams in a String

## 模式
滑动窗口 / 哈希

## 识别信号
- 找 s 中所有 p 的字母异位词起止位置
- 固定长度滑动窗口
- 频次比较

## 边界
- p 比 s 长
- p 有重复字母
- 无异位词

## 实现要点
- 维护两个哈希表频次
- p 长度的窗口滑动
- valid 计数等于字符种类数时记录

## 复杂度
- 时间: O(n)
- 空间: O(σ) σ 为字符集

## 追问
- 如果用数组代替哈希表？→ ASCII 字符集可以用数组
- 如果要求返回异位词本身？→ 记录起止位置后切片
- 与 567 题 permutation in string 的区别？→ 同样算法
