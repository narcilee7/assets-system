# LeetCode 3 — Longest Substring Without Repeating Characters

## 模式
滑动窗口 / 哈希

## 识别信号
- 连续子串无重复
- 最长长度
- 窗口收缩扩张

## 边界
- 空字符串 → 0
- 全是重复 → 1
- 全部不重复 → 整个字符串长度

## 实现要点
- 右指针扩张，左指针收缩
- 遇到重复时更新 left
- 用 Map 记录字符最新位置

## 复杂度
- 时间: O(n)
- 空间: O(min(n, σ)) σ 为字符集大小

## 追问
- 如果返回最长子串本身？→ 记录起止位置
- Unicode 字符集？→ 使用 Map 而非数组
- 如果找字母异位词？→ 416 题类似思路
