# LeetCode 76 — Minimum Window Substring

## 模式
滑动窗口 / 双指针

## 识别信号
- 最小覆盖子串
- t 中字符全部出现在子串中
- 收缩窗口条件

## 边界
- s 和 t 等长且完全匹配
- t 中有重复字符
- s 中无满足条件的子串

## 实现要点
- 维护 need Map 记录所需字符及数量
- right 扩张，left 收缩
- 收缩条件：窗口覆盖 t 中所有字符
- 更新最小窗口

## 复杂度
- 时间: O(n + m) n=s.length, m=t.length
- 空间: O(σ) σ 为字符集

## 追问
- 如果 s 和 t 只有 ASCII？→ 数组代替 Map
- 如果要求子串包含 t 的任意排列？→ 同样做法
- 如何返回子串本身？→ 记录起止位置
