# LeetCode 42 — Trapping Rain Water

## 模式
双指针 / 单调栈

## 识别信号
- 柱子间能存多少水
- 每一位置水量由 min(左边最高, 右边最高) - height 决定
- 类似木桶短板

## 边界
- 单调递减 → 0
- 单调递增 → 0
- 波浪形

## 实现要点
- 双指针：从左右向中间
- 维护 leftMax, rightMax
- 短板决定水量
- O(1) 空间

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 单调栈怎么做？→ 维护递减栈，栈顶弹出时计算
- 3D 存水？→ 二维面积更难
- 如果要返回水面形状？→ 记录每个位置水量
