# LeetCode 78 — Subsets

## 模式
回溯 / 位运算

## 识别信号
- 生成所有子集
- 每个元素可选可不选
- 2^n 个子集

## 边界
- 空数组 → [[]]
- 单元素 → [[], [a]]
- 全部选或不选

## 实现要点
- 回溯：每个位置选或不选
- 收集所有叶子节点和中间节点
- 或位运算：i 从 0 到 2^n-1，用位判断选哪些元素

## 复杂度
- 时间: O(n * 2^n)
- 空间: O(n) 递归栈

## 追问
- 位运算怎么做？→ 遍历 0..2^n-1，二进制位表示选择
- 如果数组有重复元素？→ 78题不变，90题需去重
- 与组合问题的区别？→ 组合是选固定数量，子集是选任意数量
