# LeetCode 17 — Letter Combinations of a Phone Number

## 模式
回溯

## 识别信号
- 全排列组合问题
- 每个按键对应多个字母
- 固定组合长度等于数字长度

## 边界
- 空字符串
- 数字 0 或 1（无字母映射）
- 单个数字

## 实现要点
- 回溯函数 index 表示当前处理位置
- 选择 digits[index] 对应的所有字母
- 递归到末尾收集结果

## 复杂度
- 时间: O(4^n) n 为数字长度
- 空间: O(n) 递归栈

## 追问
- 迭代怎么做？→ 逐步扩展结果集
- 如果要最短组合？→ 改成 BFS
- 如果按键映射改变？→ 修改映射表
