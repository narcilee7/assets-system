# LeetCode 131 — Palindrome Partitioning

## 模式
回溯 / DP

## 识别信号
- 字符串分割成回文子串
- 收集所有分割方式
- 分割点决策

## 边界
- 空字符串
- 全是回文
- 单字符

## 实现要点
- 回溯 + 中心扩展判断回文
- 每个位置尝试分割
- 剪枝：如果当前子串不是回文，跳过
- 用 DP 预处理 isPal[i][j] 加速判断

## 复杂度
- 时间: O(n * 2^n)
- 空间: O(n) 递归栈

## 追问
- 如何优化？→ 预计算回文 DP
- 如果只要分割数？→ 计数 DP
- 如果要找最少分割？→ 131 题类似但求 min
