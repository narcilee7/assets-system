# LeetCode 22 — Generate Parentheses

## 模式
回溯 / 动态规划

## 识别信号
- 生成所有合法的括号组合
- 卡特兰数背景
- 左右括号匹配

## 边界
- n = 0 → []
- n = 1 → ["()"]
- n = 2 → ["()()", "(())"]

## 实现要点
- 回溯：left 表示已用左括号，right 表示已用右括号
- 左括号 < n 时可加
- 右括号 < 左括号时可加（保证匹配）
- 收集到 left==right==n

## 复杂度
- 时间: O(C_n) 卡特兰数
- 空间: O(n) 递归栈

## 追问
- DP 怎么做？→ dp[i] = "(" + dp[j] + ")" + dp[i-j-1]
- 如果是 3 种括号？→ 映射扩展
- 如果要返回第 k 个组合？→ 卡特兰数公式
