# LeetCode 437 — Path Sum III

## 模式
树 / 前缀和

## 识别信号
- 二叉树中路径和为 target（不限定根到叶）
- 任意起止节点
- 前缀和思想

## 边界
- 空树 → 0
- 全负数
- target 为负数

## 实现要点
- 前缀和 + 哈希表计数
- 从根向下时记录当前路径和
- 查 target = curSum - target 在哈希表中出现次数
- 递归左右子树后回溯计数

## 复杂度
- 时间: O(n)
- 空间: O(h)

## 追问
- 根到叶路径？→ 简化，target 是叶子节点路径和
- 如何处理负数？→ 哈希表记录所有前缀和出现次数
- 双重 DFS？→ O(n^2) 不如前缀和
