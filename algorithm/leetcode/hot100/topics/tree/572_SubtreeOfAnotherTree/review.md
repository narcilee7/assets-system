# LeetCode 572 — Subtree of Another Tree

## 模式
树 / DFS / 序列化

## 识别信号
- 判断一棵树是否是另一棵树的子树
- 递归比较
- 序列化唯一表示

## 边界
- 空树 → false（题目定义）
- 完全相同
- 仅叶子节点

## 实现要点
- 序列化两棵树（前序 + null 标记）
- 子树 serialized 包含于另一棵树 serialized
- 或递归比较每个节点是否相同

## 复杂度
- 时间: O(m*n) 最坏
- 空间: O(m*n)

## 追问
- 为什么用前序？→ 唯一表示（需 null 标记）
- 后序可以吗？→ 也可以
- 如果树有重复结构？→ 必须逐节点比较
