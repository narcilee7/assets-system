# LeetCode 100 — Same Tree

## 模式
树 / DFS

## 识别信号
- 判断两棵二叉树是否完全相同
- 递归比较对应节点
- 结构和值都相等

## 边界
- 两树都为空
- 一空一非空
- 值不同

## 实现要点
- 同步遍历两棵树
- p.val === q.val && sameTree(p.left, q.left) && sameTree(p.right, q.right)
- 或用队列迭代

## 复杂度
- 时间: O(n)
- 空间: O(h)

## 追问
- 迭代怎么做？→ 队列 BFS
- 对称树和相同树的区别？→ 对称是左子树和右子树镜像
- 如果树有重复值？→ 值相等不影响
