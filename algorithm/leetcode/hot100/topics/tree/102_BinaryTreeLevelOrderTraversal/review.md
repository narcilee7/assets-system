# LeetCode 102 — Binary Tree Level Order Traversal

## 模式
BFS / 队列

## 识别信号
- 按层遍历二叉树
- 每一层单独收集
- 队列宽度优先

## 边界
- 空树 → []
- 单层树
- 只有左子树或右子树

## 实现要点
- 队列 size = 当前层节点数
- 每层单独处理
- 分层收集结果

## 复杂度
- 时间: O(n)
- 空间: O(w) w 为最大宽度，最宽 O(n)

## 追问
- 如何 BFS 返回数组？→ 已实现
- 如何 BFS 反向收集每层？→ 从右子树先入队
- 如何用 DFS 做？→ depth 参数
