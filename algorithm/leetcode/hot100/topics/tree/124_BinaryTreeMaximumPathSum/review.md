# LeetCode 124 — Binary Tree Maximum Path Sum

## 模式
树 / DFS / 动态规划

## 识别信号
- 二叉树中任意路径（节点连接）的最大和
- 路径可以从任意节点开始和结束
- 单边路径或折线路径

## 边界
- 包含负数节点
- 全负数 → 选最大负数
- 单节点

## 实现要点
- post-order DFS
- 返回值 = max(0, 单边最大) + root.val
- 全局 max = max(全局, 单边最大左 + 单边最大右 + root.val)
- 负数时保留自己（不走子节点）

## 复杂度
- 时间: O(n)
- 空间: O(h)

## 追问
- 如果路径必须是根到叶？→ 简单 DFS
- 如果找最小路径和？→ 改符号
- 如果返回具体路径？→ 记录节点
