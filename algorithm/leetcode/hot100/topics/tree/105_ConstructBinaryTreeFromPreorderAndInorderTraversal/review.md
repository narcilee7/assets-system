# LeetCode 105 — Construct Binary Tree from Preorder and Inorder

## 模式
树 / DFS / 分治

## 识别信号
- 由前序和中序遍历序列构建二叉树
- 前序第一个是根，中序根左右分割
- 递归构建左右子树

## 边界
- 前序和中序长度不匹配 → 非法输入
- 重复元素（题目一般无重复）
- 单节点

## 实现要点
- 前序 [root, ...左子树前序, ...右子树前序]
- 中序 [...左子树中序, root, ...右子树中序]
- 递归构建
- 用 Map 优化中序查找 O(1)

## 复杂度
- 时间: O(n)
- 空间: O(n) Map + 递归栈

## 追问
- 如果只有前序和后序？→ 可以，但中序不唯一
- 如何处理重复元素？→ 需要更多信息
- 如果要 O(1) 空间？→ 不可行，必须有中序或后序
