# LeetCode 236 — Lowest Common Ancestor

## 模式
树 / DFS

## 识别信号
- 二叉树中找两个节点的最近公共祖先
- 祖先可以是节点自身
- 递归向上传递信息

## 边界
- p 或 q 是根
- p 是 q 的祖先或反之
- 左右子树各找到一个

## 实现要点
- DFS 返回值：是否找到 p 或 q
- 如果左右都找到 or 当前是 p/q + 一边找到 → 当前是 LCA
- 利用返回值向上传递信息

## 复杂度
- 时间: O(n)
- 空间: O(h)

## 追问
- 如果不是二叉树（普通树）？→ 记录父指针，用 set
- 如果只有指向父节点的指针？→ 先把一个链表的祖先全存入 set，另一个找
- 如何求深度？→ 记录层数
