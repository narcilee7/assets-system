# LeetCode 114 — Flatten Binary Tree to Linked List

## 模式
树 / 迭代 / DFS

## 识别信号
- 二叉树原地转为先序遍历的链表
- 右孩子指向后继节点
- Morris 遍历或栈

## 边界
- 空树
- 只有左子树或右子树
- 叶子节点

## 实现要点
- 栈模拟递归
- 维护 prev 指针反向链接
- 或 Morris 遍历 O(1) 空间

## 复杂度
- 时间: O(n)
- 空间: O(h)

## 追问
- Morris 遍历怎么做？→ 线索二叉树
- 为什么不用递归？→ 递归本身是 O(h) 空间
- 如果要把结果存到右子树而不是新链表？
