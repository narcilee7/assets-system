# LeetCode 543 — Diameter of Binary Tree

## 模式
树 / DFS

## 识别信号
- 二叉树最长路径（边数）
- 路径不一定经过根
- 左右子树深度和

## 边界
- 空树 → 0
- 单边树
- 完全平衡

## 实现要点
- DFS 返回深度
- 全局 diameter 更新：左深度 + 右深度
- 深度 = max(左, 右) + 1
- 取 max 而非 sum

## 复杂度
- 时间: O(n)
- 空间: O(h)

## 追问
- 直径是边数还是节点数？→ 边数，124 题是节点数
- 如果要返回路径？→ 记录直径端点
- 如何处理非二叉树？→ 图论，树的直径用 BFS
