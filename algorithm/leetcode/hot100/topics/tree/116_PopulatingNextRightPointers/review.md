# LeetCode 116 — Populating Next Right Pointers

## 模式
BFS / 迭代 / 常数空间

## 识别信号
- 将完美二叉树的每层节点用 next 连接
- 层序连接，不使用队列
- O(1) 空间

## 边界
- 空树
- 单层树
- 最后一层节点 next 指向 null

## 实现要点
- 利用已建立的 next 指针遍历父节点
- 左孩子 next = 右孩子
- 右孩子 next = 父节点.next 的左孩子

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 如果不是完美二叉树？→ 117 题
- 如果用队列做？→ O(n) 空间
- 如果要求返回根节点？→ 已连接
