# LeetCode 155 — Min Stack

## 模式
栈 / 设计

## 识别信号
- 栈 plus 支持 O(1) 获取最小值
- 普通栈无法 O(1) 获取 min
- 辅助栈或差值存储

## 边界
- 空栈
- 单一元素
- 连续 pop 最小值

## 实现要点
- 辅助栈同步 push/pop
- 辅助栈顶 = 当前栈内最小值
- 或差值法：存 (val, val-minSoFar)

## 复杂度
- 时间: O(1) 所有操作
- 空间: O(n)

## 追问
- 如果要 O(1) 空间？→ 差值法，但有精度问题
- 如果要 O(1) 获取中位数？→ 辅助栈
- 单调栈？→ 维持递减栈
