# LeetCode 394 — Decode String

## 模式
栈

## 识别信号
- 解码重复模式字符串
- 3[a2[bc]] → aaabcbcbc
- 嵌套括号

## 边界
- 无数字
- 多位数如 12[a]
- 连续括号

## 实现要点
- 数字栈和字符串栈
- 遇到 [ 入栈当前字符串，重置
- 遇到 ] 弹出数字和之前字符串，重复拼接到之前字符串
- 累积字符串不用每次 join

## 复杂度
- 时间: O(n * max_k)
- 空间: O(n)

## 追问
- 递归怎么做？→ 找到匹配的 [] 递归处理
- 如果有嵌套且多层？→ 栈模拟递归
- 如果要求 in-place？→ 不可能，输出更长
