# LeetCode 122 — Best Time to Buy and Sell Stock II

## 模式
贪心

## 识别信号
- 允许多次交易
- 最大化利润
- 只要第二天比今天高就买入卖出

## 边界
- 单日或价格下跌 → 0
- 连续上涨
- 全年单边下跌

## 实现要点
- 累加所有上涨差值：sum(max(0, prices[i] - prices[i-1]))
- 等价于低买高卖但分割成无数个小交易

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 为什么贪心最优？→ 利润可加，任何分割不改变总和
- 如果有手续费？→ DP
- 如果限制交易次数？→ 分段 DP
