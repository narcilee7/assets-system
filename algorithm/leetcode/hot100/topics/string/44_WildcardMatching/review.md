# LeetCode 44 — Wildcard Matching

## 模式
贪心 / DP

## 识别信号
- ? 匹配任意单个字符
- * 匹配任意序列
- 比 10 正则简单

## 边界
- 全是 *
- * 连续或不连续
- ? 和 * 混合

## 实现要点
- 双指针贪心
- i 指向 s，j 指向 p 中 * 位置
- 遇到 * 记录 starI 和 starJ
- 不匹配时用记录的 * 回溯

## 复杂度
- 时间: O(mn) 最坏，但实际 O(m+n)
- 空间: O(1)

## 追问
- DP 怎么做？→ dp[i][j] = s 前 i 个匹配 p 前 j 个
- ? 和 * 的区别？→ ? 只匹配一个，* 匹配任意
- 为什么贪心可行？→ * 可匹配任意，回溯找最优匹配
