# LeetCode 647 — Palindromic Substrings

## 模式
中心扩展 / 动态规划

## 识别信号
- 计算回文子串个数
- 连续子串
- 中心扩展 O(n^2)

## 边界
- 空字符串 → 0
- 全相同字符
- 单字符

## 实现要点
- 中心扩展：每个位置向两边扩展
- 奇数长和偶数长分别处理
- 以每个字符为中心的扩展次数即新增回文数

## 复杂度
- 时间: O(n^2)
- 空间: O(1)

## 追问
- Manacher 算法？→ O(n) 但较复杂
- DP 怎么做？→ dp[i][j] = dp[i+1][j-1] + 1 if s[i]==s[j]
- 最长回文子串？→ 记录起止位置
