# LeetCode 21 — Merge Two Sorted Lists

## 模式
链表 / 双指针 / 归并

## 识别信号
- 两个有序链表合并
- 类似归并排序的 merge 步骤
- 虚拟头节点简化边界处理

## 边界
- 任意链表为空
- 两链表都为空

## 实现要点
- 虚拟头节点 dummy 简化操作
- 每次取较小者接入结果
- 最后拼接剩余部分

## 复杂度
- 时间: O(n+m)
- 空间: O(1) 迭代，O(n+m) 递归栈

## 追问
- 如果要 in-place 不能用新节点？→ 直接修改指针
- K 个有序链表？→ 优先队列
- 如果链表无序呢？→ 先排序
