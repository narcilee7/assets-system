# LeetCode 141 — Linked List Cycle

## 模式
快慢指针（Floyd 判圈）

## 识别信号
- 判断链表是否有环
- 无法用额外存储
- 空间要求 O(1)

## 边界
- 空链表 → false
- 单节点且不指向自身 → false
- 环在起始位置

## 实现要点
- slow 每次走 1 步，fast 每次走 2 步
- 有环必相遇
- 无环 fast 先到 null

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 如何找到环的入口节点？→ 142 题
- 为什么 fast 走 2 步而不是 3 步？
- 如何在不使用额外空间下检测？→ 标记法（修改节点不可行则无解）
