# LeetCode 2 — Add Two Numbers

## 模式
链表 / 数学

## 识别信号
- 两数反向存储在链表
- 逐位相加
- 链表中节点是数字位

## 边界
- 进位到超出链表长度
- 任一链表为空
- 结果链表首位需要进位

## 实现要点
- 遍历两链表，逐位相加 carry
- 新节点存储 (sum % 10)
- 最后一节点若 carry > 0 则新增节点

## 复杂度
- 时间: O(max(m,n))
- 空间: O(max(m,n))

## 追问
- 如果链表正向存储？→ 先反转
- 如果要 O(1) 空间？→ 复用较长链表的节点
- 如果是大数相加？→ 链表本身就是大数表示
