# LeetCode 142 — Linked List Cycle II

## 模式
快慢指针（Floyd 判圈扩展）

## 识别信号
- 找到链表环的入口节点
- 141 题扩展
- 数学推导

## 边界
- 无环 → null
- 环在头节点
- 环很小

## 实现要点
- 相遇时：slow 走 k 步，fast 走 2k 步，设环长 r，相遇点距入口 a
- 2k - k = nr → k = nr
- 从 head 和相遇点同时走，相遇点即入口

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 如何证明数学关系？→ 设起点到入口距离为 a，slow 入环时 fast 已领先
- 无解情况？→ 永远不相遇
- 如何求环长？→ 从入口走一圈回到入口
