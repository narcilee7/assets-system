# LeetCode 287 — Find the Duplicate Number

## 模式
快慢指针 / 二分

## 识别信号
- 数组值在 [1,n]，必有一个重复
- 不能修改数组
- O(1) 空间

## 边界
- 重复两次以上
- 重复数字可以是任何值
- 单个重复 vs 多重重复

## 实现要点
- 快慢指针：数组值作为 next 指针，形成环入口即是重复
- 二分：count <= mid 判断重复数量，在 [1,mid] 或 [mid+1,n]
- 快慢指针：相遇后从起点再走直到相遇

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 二分和快慢哪个更优？→ 快慢 O(n) 但不用额外空间
- 为什么有环？→ 重复值导致两指针指向同一节点
- 如果值不在 [1,n]？→ 哈希表
