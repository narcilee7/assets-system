# LeetCode 25 — Reverse Nodes in k-Group

## 模式
链表

## 识别信号
- 每 k 个节点一组反转
- 组内反转，组间连接
- 不足 k 个不反转

## 边界
- 最后一组不足 k 个
- k = 1 或 k >= 链表长度
- 空链表

## 实现要点
- 计数 k 个节点
- 反转 k 个节点（206 题思路）
- 递归或迭代连接
- 记录下一组头节点

## 复杂度
- 时间: O(n)
- 空间: O(1) 迭代，O(n/k) 递归栈

## 追问
- 递归怎么做？→ 先反转当前组，再递归处理下一组
- 如果要返回新头？→ 记录反转后的新头
- 空间 O(1) 怎么做？→ 迭代需维护 prev, curr, next 指针
