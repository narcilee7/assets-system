# LeetCode 297 — Serialize and Deserialize Binary Tree

## 模式
设计 / DFS / BFS

## 识别信号
- 二叉树序列化
- 字符串和树结构互转
- 先序/层序均可

## 边界
- 空树
- 单节点
- 极端不平衡树

## 实现要点
- 序列化：先序遍历，空节点用 # 标记，节点间用分隔符
- 反序列化：从字符串重建
- BFS 层序：队列，空节点也要记录

## 复杂度
- 时间: O(n)
- 空间: O(n)

## 追问
- 如果要求压缩？→ 前序遍历比层序省分隔符
- 如果树有重复值？→ 分隔符必须能区分
- JSON 格式？→ 同样原理
