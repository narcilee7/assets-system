# LeetCode 169 — Majority Element

## 模式
投票法（Boyer-Moore）

## 识别信号
- 出现次数 > n/2 的元素（众数）
- 必定存在（题目保证）
- 要求 O(1) 空间

## 边界
- 数组非空
- 众数一定存在

## 实现要点
- candidate 票数抵消思想
- 票数为零时更换候选人
- 最终 candidate 即为众数

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 如果不存在众数（出现 > n/2）呢？→ 需要验证
- 如果找出现 > n/3 的数？→ 摩尔投票扩展
- 如果要求返回下标？→ passes 验证阶段
