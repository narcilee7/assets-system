# LeetCode 136 — Single Number

## 模式
位运算

## 识别信号
- 数组中除一个元素外都出现两次
- 找那个只出现一次的
- O(1) 空间

## 边界
- 只有一个元素
- 负数
- 最大/最小值

## 实现要点
- 异或性质：a^a=0, a^0=a, 满足交换律结合律
- 所有元素异或，结果即 single number
- 出现两次的抵消

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 追问
- 如果两个 single number？→ 分组异或（260题）
- 如果有三个 single number？→ 困难
- 如果要求不用异或？→ 哈希表 O(n) 空间
