# Systems Case Studies

案例用于把系统工程能力串起来。

## 推荐案例

| 案例 | 关联能力 |
| --- | --- |
| Slow API Diagnosis | Linux、network、database、observability |
| High QPS Gateway | load balancing、rate limit、cache、SLO |
| Object Storage Upload | multipart、checksum、consistency、retry |
| Queue Backlog Incident | messaging、worker、autoscaling、SLO |
| Kubernetes Pod CrashLoop | container、resource、probe、logs |
| Database Replication Lag | transaction、replication、read consistency |
| AI Streaming Platform | SSE、backpressure、tool runtime、observability |
